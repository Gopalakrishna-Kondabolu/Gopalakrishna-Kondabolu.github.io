How to Use

1. Clone the repo:
   git clone https://github.com/Gopalakrishna-Kondabolu/Gopalakrishna-Kondabolu.github.io.git
   Make your changes locally.

Commit and push:

git add .
git commit -m "Update site"
git push
Your changes will automatically reflect on your GitHub Pages website in a few seconds.
